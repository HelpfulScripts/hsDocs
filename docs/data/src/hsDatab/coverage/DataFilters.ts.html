* ```
*

* ```
* RecursiveCondition =
*    OrCondition               -> OR: true if any compound condition is true

*
* SetAndCondition = {          -> AND: true if all sub-conditions are true
*    'or':  RecursiveCondition    -> true if any RecursiveCondition is true
