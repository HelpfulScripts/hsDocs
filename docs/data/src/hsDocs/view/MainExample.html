   1/**

   2 * Comment sections may contain code examples that are intended to be live demonstrations for 

   3 * explaining some concept of the module are contained in. 

   4 * 

   5 * `hsDocs` will interpret comment sections between `` and ``

   6 * tags as code examples. Inside a code example, `'script.js'>` tags contain a script to be executed, 

   7 * and `'style.css'>` tags contain stylesheet instructions to apply, as in the following snippet (also see Example below):

   8 * 

   9 *     <example> 

  10 *     'script.js'>

  11 *     m.mount(root, { 

  12 *         view:() => m(hsLayout.Layout, { columns:[], 

  13 *             content:['first line','second line')]

  14 *         })

  15 *     });

  16 *     

  17 * 

  18 *     'style.css'>

  19 *     .hsLeaf { 

  20 *         color: blue; 

  21 *     }

  22 *     

  23 *     </example>

  24 *  

  25 * 

  26 * ### Scripts 

  27 * Scripts are enclosed by `'script.js'>` and `` tags. 

  28 * 

  29 * hsDocs internally uses the [global Function object][Function] to parse and execute the script. 

  30 * Thus the script runs in the global scope and has access to global objects. A number of additional modules are provided

  31 * as parameters in the `Function` constructor and are thus available to the script. 

  32 * hsDocs currently provides the following default namespaces to access the corresponding modules in the scripts:

  33 * - **m**: namespace for the [`Mithril`](https://mithril.js.org/) m function    

  34 * - **hsLayout**: namespace for the [`hsLayout`](http://helpfulscripts.github.io/hsLayout/#!/api/hsLayout/0) library, 

  35 * providing functions to layout the browser window.

  36 * - **hsWidget**: namespace for the [`hsWidget`](https://helpfulscripts.github.io/hsWidget/#!/api/hsWidget/0) library, 

  37 * providing various UI widget functions.

  38 * - **root**: the DOM element to attach content to.

  39 * 

  40 * Additional Scripts can loaded - see `Configuring the example` below. 

  41 * 

  42 * The script can access and manipulate any of the provided libraries

  43 * via their namespace, for example as in `m(hsLayout.Layout,{})`

  44 * #### Using Mithril:

  45 * mount a `mithril Vnode` on the provided root DOM element using `m.mount` or `m.render`. 

  46 * Do not use `m.route` as only a single call is allowed per web app and that is used to manage the 

  47 * main hsDocs site menu and navigation.

  48 * ```

  49 * m.mount(root, { view:() => m('div', 'content')});

  50 * ```

  51 * #### Using d3:

  52 * ```

  53 * const base = d3.select(root);

  54 *  const svg = base.append('svg')

  55 *     .classed('baseSVG', true)

  56 *     .attr('height', '100%')

  57 *     .attr('width', '100%')

  58 *     .attr('preserveAspectRatio', 'xMinYMin meet');

  59 * ```

  60 * 

  61 * ### Configuring the `example`

  62 * The `` accepts additional arguments to configure the example:

  63 * 

  64 * #### Height of the `example` panel

  65 * To change the default panel height of 300px, set the `height=px` argument.

  66 * 

  67 * #### Module injection

  68 * Additional module libraries can be injected by adding the argument `libs={:[,...]}`.

  69 * `hsDocs` will try to load the library via the provided `path` and make it available to the script within the namespace `export name`. 

  70 * For example:

  71 * 

  72 * 'hsGraphd3', d3:'https://d3js.org/d3.v5.min.js'}>

  73 * ...in the script:

  74 *    new hsGraph.Graph();

  75 *    d3.slect('');

  76 * 

  77 * The following rules apply in resolving `path`:

  78 * 1. If `path` is structured, i.e. contains '/', it is requested as is - see 'd3' example above.

  79 * 2. load from `node_modules//.js`, relative to the current web page. 

  80 * This is useful to maintain libraries installed via `npm` locally on the server.

  81 * 3. load from `https://helpfulscripts.github.io//.js`. Provided as a convenience to access `helpfulscript` modules ;)

  82 *  

  83 * To be recognized as injectable module, libraries need to follow the convention of `webPack's` 

  84 * [`output.libraryTarget: "this"`](https://webpack.js.org/configuration/output/#expose-via-object-assignment) configuration.

  85 * 

  86 * [Function]:https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function

  87 * 

  88 * ### Styles

  89 * Styles will be automatically sandboxed so they are valid only within the enclosing example tags. 

  90 * This is achieved by prefixing css tags with a unique exmple ID and allows multiple examples to co-exist on the same page.

  91 * In the DOM, the example ID is assgned top the <example> tag.

  92 * Use `$exampleID` as css tag in the `css` section of the example to refer to the <example> element, 

  93 * as shown below.

  94 * 

  95 * ### Example

  96 * 

  97 * script.js'>

  98 * m.mount(root, { 

  99 *     view:() => m(hsLayout.Layout, {

 100 *         css:'.myExample', 

 101 *         columns:[], 

 102 *         content:[

 103 *             'left column',

 104 *             'right column'

 105 *         ]

 106 *     })

 107 * });

 108 * 

 109 * style.css'>

 110 * $exampleID { height: 200px;}

 112 *     margin:0; 

 113 * }

 114 * .myExample {

 165/**

 166 * creates the example configuration, generates the DOM hook, and sets up the example execution.

 167 * `example` takes a context map of the form `{ name:module, ...}` containing libraries to be used 

 168 * within the example, and returns a function that can be used in calls to `string.replace`. 

 169 * See the following example:

 170 * 



 171 * import * as layout from "layout";


 172 * text = text.replace(/([\S\s]*?)<\/example>/gi, example({layout:layout}));


 173 * 

 174 * The modules `m`, `hsLayout`, and `hsGraph` will be added by default as 

 175 * ` { m: m, layout: layout, widget: widget } `

 176 * @param exmpl the example string extracted from the comment, including the `` tags.

 177 * @param context the context in which the example script should be run, expressed as `name`:`namespace` pairs.

 178 */

 179export function example(exmpl:string) { 

 180    const instance = shortCheckSum(exmpl);

 181    let IDs = gInitialized[instance]; 

 182    if (!IDs) {

 183        IDs = gInitialized[instance] = initDesc(IDs);

 184        IDs.executeSource = exmpl;

 185        createExecuteScript(IDs, exmpl);

 186    }

 187    if (!document.getElementById(IDs.menuID)) { 

 188        addExampleStructure(IDs);

 189        delay(1)(IDs)

 190            .then(executeScript)

 191            .catch(executeError);

 192    }

 193

 194    const frameHeight = (IDs.attrs? IDs.attrs.height : undefined) || '300px';

 195    const wrapWithID = (css:string) => css==='$exampleID'? `#${IDs.exampleID}`: `#${IDs.menuID} ${css}`;

 196

 197    // prefix css selectors with ID of main execution area to sandbox the scope

 198    // (^|}): start of string or end of previous style def

 199    // \s*?: any white spaces

 200    // (\S*?): capturing group: css name

 201    // \s*?{: whitespaces, followed by start of style def

 202    const style = (!IDs.pages['css'])? '':                              // empty if no css defined

 203                    IDs.pages['css'].replace(/(^|})\s*?(\S*?)\s*?{/gi,    // otherwise wrap each css statement

 204                        (x:string, ...args:any[]) => x.replace(args[1], wrapWithID(args[1]))

 205    );

 206    return `"height:${frameHeight}" class="hs-layout-frame">`;

 207}

 209async function createExecuteScript(IDs:CommentDescriptor, exmpl:string): Promise {

 210    try {

 211        const desc = await getCommentDescriptor(IDs, exmpl);

 212        IDs.runScript = (root:any) => {

 213            const libNames = Object.keys(modules);

 214            const missingLibs = IDs.attrs.libs.map(l => l.sym).filter(l => libNames.indexOf(l)<0);

 215            if (missingLibs.length > 0) {

 216                log.info(`expected modules to be loaded: ${missingLibs.join(', ')}`);

 217            } else {

 218                const libs = libNames.map(name => modules[name]);

 219                const scriptFn = new Function('root', ...libNames, desc);    

 220                log.info('running example script');

 221                scriptFn(root, ...libs);

 222            }

 223        };

 224        log.debug('example function ready');

 225        IDs.created = true;

 226        return true;

 227    } catch(e) { 

 228        log.error(`creating script: ${e}\n${e.stack}`); 

 229        return false;

 230    } 

 231}

 299/**

 300 * execute the provided script 

 301 * @param IDs provides the context in which the script is exceuted/

 302 */

 303function executeScript(IDs:CommentDescriptor) {

 304    const root = document.getElementById(IDs.menuID);

 305    if (root && IDs.created) {

 306        // root.querySelectorAll('.placeholder').forEach(d => root.removeChild(d));

 307        while (root.children.length>0) { root.removeChild(root.children[0]); }

 308        try { IDs.runScript(root); }

 309        catch(e) { 

 310            log.error(`executing script: ${e}\n${IDs.executeSource}\n${e.stack}`); 

 311        }

 312    } else {

 313        // log.error(`root not found for menuID ${IDs.menuID}`);

 314    }

 315    m.redraw();

 316    return IDs;

 317}

 323/**

 324 * Fills in all fields of the CommentDescriptor provided as `IDs`.

 325 * @param IDs the CommentDescriptor to complete

 326 * @param example the example string, including  tag

 327 * @return the script to execute, as a string

 328 */

 329async function getCommentDescriptor(IDs:CommentDescriptor, example:string):Promise {

 330    let result = '';

 331    

 332    // let attrs = example.match(/)/gi);

 333    let attrs = example.match(//); // capture all parameters to `example`

 334    if (attrs && attrs[1]) { 

 335        findTokens(IDs, attrs[1]);

 336        if (IDs.attrs.libs.length>0) {

 337            await Promise.all(IDs.attrs.libs.map(async l => {

 338                log.debug(`loading lib: ${l.sym}: ${l.path}`);

 339                const content = await loadScript(l.sym, l.path);

 340                if (content) { modules[l.sym] = content[l.sym]; }

 341            }));

 342        }

 343        log.debug(`found attrs:\n ${log.inspect(IDs.attrs, 5)}`);

 344    }

 345    example.replace(/|"]>([\S\s]*?)<\/file>/gi, function(text:string) {

 346        const args = [...arguments];

 347        const content = args[2].trim(); // the part between  and 

 348        IDs.desc.items.push(args[1]);   // record available extensions, i.e. 'js', 'html', etc 

 349        IDs.pages[args[1]] = content;   // associate the content with the extension

 350        return result;

 351    });

 352    return IDs.pages['js'];

 353}

 355function findTokens(IDs:CommentDescriptor, str:string) {

 356    const results = [];

 357    // find 'key=value' and 'key={value}' statements

 358    const reg = new RegExp(/((\w+?)\s*?=\s*?([^\s{},]+|{[\s\S]+?}))/g);

 359    let r;

 360    while(r = reg.exec(str)) {

 361        results.push({key:r[2], val:r[3]});

 362    }

 363    results.map(kv => {

 364        switch (kv.key) {

 365            case 'height': IDs.attrs['height'] = kv.val; break;       

 366            case 'libs':    // {graph:'/node_modules/hsgraphd3/index.js'}

 367                const ligreg = new RegExp(/((\w+?)\s*:\s*([^\s,}]+))/g);

 368                while(r = ligreg.exec(kv.val)) {

 369                    IDs.attrs.libs.push({sym:r[2],path:r[3].replace(/['"]/g, '')});

 370                }

 371        }

 372    });

 373    log.debug(`${str} --> \n${log.inspect(IDs.attrs,5)}`);

 374}

 377/**

 378 * attempts to load a library for the specified `path`. If `path` is a module nane (e.g. 'hsDocs'), 

 379 * this involves the following attempts

 380 * at resolving the library path:

 381 * 1. load from 'node_modules//.js, relative to the current web page.

 382 * 2. load from 'https://helpfulscripts.github.io//.js'

 383 * If `path` is structured, i.e. contains '/', it is requested as is. 

 384 * @param path module name, such as 'hsDocs'

 385 */

 386async function loadScript(sym:string, path:string) {

 387    const load = (p:string) => m.request({ method: "GET", url: p, extract: (xhr:any) => xhr });

 388

 389    const paths = [

 390        `node_modules/${path.toLowerCase()}/${path}.js`,

 391        `https://helpfulscripts.github.io/${path}/${path}.js`

 392    ];

 393    let content:any;

 394    try {

 395        if (path.indexOf('/')>=0) {     // if structured: call as is

 396            content = await load(path);

 397        } else {                        // else try other paths

 398            for (let i=0; i<=paths.length; i++) {

 399                try { 

 400                    path = paths[i];

 401                    content = await load(paths[i]); 

 402                    if (content.status === 200) { break; }

 403                }

 404                catch(e) {}

 405            }

 406            log.info(`${path} ${content?'found':'not found'}`);

 407        }

 408    } catch(e) { log.error(`loading lib ${path}`);}

 409    let code:string;

 410    try { 

 411        code = content.responseText;

 412        let lib:any = {};

 413        log.info(`loaded ${path}, creating library`);

 414        new Function(code).bind(lib)();

 415        if (lib) {

 416            return lib;

 417        } else {

 418            log.warn(`wrong lib format for ${sym}: ${code.slice(0, 20)}..., should be 'this["${sym}"] = ...`);

 419        }

 420    } catch(e) {

 421        log.error(`JSON.parse: ${path}: ${e}`);

 422        log.error(code);

 423        return undefined;

 424    }

 425    // add(content.responseText);

 426}
